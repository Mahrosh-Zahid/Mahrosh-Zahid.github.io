Web Scraping

A A A

Tools Used: Python - Beautiful Soup, urllib, MechanicalSoup, Requests


I wrote a Python script that uses the urllib.request library to fetch the HTML content of a webpage. Specifically, I targeted the profile page of Aphrodite from the Real Python Olympus website. After retrieving the HTML content, I decoded it to a readable format using UTF-8 encoding.

Then, I used regular expressions (re module) to extract specific information from the HTML. For instance, I extracted the title of the webpage ("Profile: Aphrodite") and the name ("Aphrodite") using regex patterns.

Additionally, I employed the BeautifulSoup library to parse the HTML content and extract text without HTML tags. This helped me get cleaner text data for further processing.

Later in the code, I utilized MechanicalSoup to interact with web forms. I logged into a website by submitting a login form with predefined username and password values. Then, I extracted information from another webpage using CSS selectors to locate specific elements.

This tutorial served as a valuable resource and guided me through the process of learning web scraping techniques in Python: https://realpython.com/python-web-scraping-practical-introduction/.
